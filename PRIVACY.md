# Privacy Policy for Danbooru Enhanced

**Last Updated: April 18, 2026**

Danbooru Enhanced (the "Extension") is committed to protecting your privacy. This Privacy Policy explains how we handle your information.

## 1. No Data Collection
**We do not collect, store, or transmit any personal information or usage data.**
- The Extension does not have a backend server.
- The Extension does not use tracking or analytics cookies.
- No data is sent to any third parties.

## 2. Interaction with Danbooru
The Extension only interacts with the Danbooru website (`danbooru.donmai.us`) to provide its features:
- **Quick Filters**: Modifies the search tag input field on the page.
- **Masonry Layout**: Adjusts the CSS and DOM structure of the image gallery.
- **High-Resolution Images**: Fetches image metadata directly from the Danbooru API (`/posts.json`) to display higher-quality previews. 

All network requests are made directly to the Danbooru server.

## 3. Local Storage
The Extension uses your browser's `localStorage` to save your layout preference (whether Masonry mode is enabled or disabled). This data remains on your local machine and is never uploaded.

## 4. Permissions
- **activeTab**: Required to interact with the current Danbooru page you are visiting to inject the user interface and apply styles.

## 5. Changes to This Policy
We may update this Privacy Policy from time to time. Any changes will be reflected in this document.

## 6. Contact
If you have any questions, please open an issue on the GitHub repository.

# Danbooru Enhanced 隐私政策

**最后更新：2026年4月18日**

Danbooru Enhanced（“本扩展”）致力于保护您的隐私。本隐私政策说明了我们如何处理您的信息。

## 1. 不收集数据
**我们不收集、存储或传输任何个人信息或使用数据。**
- 本扩展没有后端服务器。
- 本扩展不使用追踪或分析 Cookie。
- 不会向任何第三方发送任何数据。

## 2. 与 Danbooru 的交互
本扩展仅与 Danbooru 网站 (`danbooru.donmai.us`) 进行交互以提供其功能：
- **快捷过滤**：修改页面上的搜索标签输入框。
- **瀑布流布局**：调整图片库的 CSS 和 DOM 结构。
- **高清图片**：直接从 Danbooru API (`/posts.json`) 获取图片元数据以显示更高质量的预览。

所有网络请求都直接发送到 Danbooru 服务器。

## 3. 本地存储 (Local Storage)
本扩展使用浏览器的 `localStorage` 来保存您的布局偏好（是否启用瀑布流模式）。此数据存储在您的本地机器上，永远不会被上传。

## 4. 权限说明
- **activeTab**：需要此权限以与您正在访问的 Danbooru 页面进行交互，从而注入用户界面并应用样式。

## 5. 政策修订
我们可能会不时更新此隐私政策。任何更改都将反映在此文档中。

## 6. 联系方式
如果您有任何疑问，请在 GitHub 仓库中提交 Issue。
